Sobre o Projeto

O AcessoEdu Nordeste é uma plataforma de transparência e auditoria cidadã voltada à infraestrutura das escolas públicas do Nordeste brasileiro. Os dados são extraídos dos microdados oficiais do Censo Escolar (INEP) e apresentados de forma visual, interativa e acessível a qualquer cidadão.

APIs Externas

BrasilAPI

API pública de consulta de endereço por CEP. Utilizada para a busca integrada por CEP no dashboard e como fallback na página de detalhes da escola para buscar logradouro e bairro quando o endereço não está disponível no banco de dados principal.

Bibliotecas e Front-end

Chart.js

Biblioteca gráfica de alta performance em canvas. Utilizada para gráficos de barras comparativos, rosca (doughnut), radar e evolução temporal.

TailwindCSS

Framework CSS utilitário para estilização responsiva. Todos os componentes da interface utilizam classes Tailwind, garantindo consistência visual e design system unificado.

Vanilla JS (ES6 Modules)

JavaScript puro com módulos ES6, implementando arquitetura Pub/Sub via Event Bus (estado.js) para reatividade entre componentes. Separação estrita entre camadas de API, UI e núcleo.

Back-end e Dados

Back4App (Parse Server)

Backend como Serviço (BaaS) responsável por autenticação de usuários, banco de dados NoSQL, armazenamento de arquivos (fotos) e consultas geoespaciais. Classes principais: Escolas2024, Escolas2025, EstatisticasAgregadas, Avaliacoes, SchoolPhoto.

Scripts de ETL em Python

Mineração e processamento dos microdados do INEP utilizando Python 3.10+ com Pandas. Os CSVs oficiais do Censo Escolar são extraídos, normalizados e convertidos em JSON para importação no Back4App via seeders Node.js.
