Como é calculada esta nota?

O Índice de Excelência em Infraestrutura é calculado a partir de 7 indicadores binários do Censo Escolar INEP:

  • Internet
  • Biblioteca
  • Laboratório de Informática
  • Quadra de Esportes
  • Rampas de Acessibilidade
  • Banheiro Adaptado PCD
  • Água Potável

A nota final é a média simples dos 7 indicadores (cada um vale 0 ou 1), normalizada para escala de 0 a 10.

Ouro: 9.0+ Prata: 7.5+ Bronze: 6.0+
