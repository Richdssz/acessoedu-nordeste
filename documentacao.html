Documentação do Projeto

AcessoEdu Nordeste — Plataforma de transparência e auditoria cidadã para infraestrutura escolar do Nordeste brasileiro.

Fontes de Dados

Censo Escolar INEP

Os dados de infraestrutura escolar são extraídos dos microdados oficiais do Censo Escolar da Educação Básica, disponibilizados pelo INEP (Instituto Nacional de Estudos e Pesquisas Educacionais Anísio Teixeira) para os anos de 2024 e 2025.

Indicadores binários de infraestrutura (Internet, Laboratório, Banheiro PNE, Quadra, Acessibilidade, Água Potável, Energia Elétrica)
Dados de geolocalização (latitude e longitude) das escolas
Dependência administrativa (Federal, Estadual, Municipal, Privada)
Telefone e endereço das escolas
Amostra: ~74 mil escolas nos 9 estados do Nordeste (2024 e 2025)

Back-end / Dados

Back4App (Parse Server)

BaaS (Backend as a Service) que armazena e serve os dados escolares, avaliações e fotos. Utiliza o SDK Parse para consultas com GeoPoint e controle de acesso baseado em Roles (admin/user). As tabelas Escolas2024 e Escolas2025 contêm ~74 mil registros cada, com 7 indicadores binários de infraestrutura mais metadados.

Pipeline de Dados (Python + Node.js)

Scripts Python extraem e tratam os microdados do INEP (CSV), gerando JSONs intermediários. Seeders Node.js importam os dados para o Back4App em lotes com retry e health-check. Scripts de PATCH atualizam campos complementares (dependência, número, telefone) sem reimportação completa.

APIs Externas

BrasilAPI (Busca de CEP e Logradouro)

API pública brasileira que fornece dados cadastrais do CEP. Utilizada para a busca integrada de escolas por município a partir do CEP informado e como fallback de endereço completo (logradouro, bairro, cidade e estado) na página de detalhes da escola. Gratuita e sem autenticação.

Bibliotecas / Front-end

Chart.js

Biblioteca leve para gráficos em canvas. Utilizada em gráficos de barras comparativos 2024/2025, donut de conectividade, radar de infraestrutura (escola vs médias regionais) e linha de evolução. Carregada sob demanda (lazy load) via IntersectionObserver para não bloquear o carregamento inicial da página.

TailwindCSS

Framework CSS utilitário carregado via CDN. Fornece estilização responsiva com classes atômicas, tema customizado com as cores institucionais do projeto (primária, secundária, acento, mobilidade) e fontes Inter e Poppins.

Phosphor Icons

Biblioteca de ícones open source com mais de 1.300 ícones em 6 estilos (thin, light, regular, bold, fill, duotone). Utilizada em toda a interface para iconografia consistente: navegação, indicadores, botões e badges.

Vanilla JS (ES6 Modules)

Lógica da aplicação em JavaScript puro com módulos ES6, sem frameworks. Arquitetura Pub/Sub com estado centralizado (estado.js), separação em camadas (api/, ui/, core/) e lazy loading de dependências pesadas.
