Documentação do Projeto

AcessoEdu Nordeste — Plataforma de transparência e auditoria cidadã para infraestrutura escolar do Nordeste brasileiro.

Fontes de Dados

Censo Escolar INEP

Os dados de infraestrutura escolar são extraídos dos microdados oficiais do Censo Escolar da Educação Básica, disponibilizados pelo INEP (Instituto Nacional de Estudos e Pesquisas Educacionais Anísio Teixeira) para os anos de 2024 e 2025.

Indicadores binarios de infraestrutura (Internet, Biblioteca, Laboratorio, Quadra, Rampas, Banheiro Acessivel, Água Potável)
Dados de geolocalizacao (latitude e longitude) das escolas
Niveis de ensino ofertados (Infantil, Fundamental, Medio, EJA, Especial, Profissionalizante)
Dependência administrativa (Federal, Estadual, Municipal, Privada)
Quantidade de psicólogos por escola
Amostra: 150 escolas comuns aos dois anos (2024 e 2025)

APIs Integradas

Back4App (Parse Server)

BaaS (Backend as a Service) que armazena e serve os dados escolares, avaliações e fotos. Utiliza o SDK Parse para consultas geoespaciais com GeoPoint, buscas por raio de proximidade, e controle de acesso baseado em Roles (admin/user).

ViaCEP

API pública brasileira para consulta de endereços por CEP. Integrada ao modal "Perto de Mim" do dashboard, permitindo que o usuário digite um CEP e tenha os filtros de Estado e Município preenchidos automaticamente. A resposta inclui logradouro, bairro, cidade e UF.

Mapillary Graph API

API de imagens de rua geolocalizadas (street-level imagery). Utilizada como fallback na cascata de imagens da página da escola: busca fotos da fachada através de uma bounding box de 0.001 graus ao redor das coordenadas da escola, retornando até 5 imagens com resolução de 1024px para o carrossel.

Leaflet + OpenStreetMap

Biblioteca Leaflet.js com tiles do OpenStreetMap para renderização do mapa interativo no dashboard. Inclui o plugin MarkerCluster para agrupamento de marcadores em zoom distante, melhorando a performance com centenas de escolas plotadas simultaneamente.

Chart.js

Biblioteca leve para gráficos em canvas. Utilizada no gráfico radar da página da escola, comparando a infraestrutura da escola com as médias do município, estado e região em 7 indicadores simultâneos. Carregada sob demanda (lazy load) via IntersectionObserver.
