Como usar os filtros

Use os filtros de Estado e Município no dashboard para restringir a visualização a uma região específica. O filtro de Ano alterna entre os dados de 2024 e 2025. O campo de Busca permite localizar escolas pelo nome.

O que é o Indicador Geral

O Indicador Geral é uma nota de 0 a 10 calculada a partir de 7 indicadores binários de infraestrutura: Internet, Biblioteca, Laboratório de Informática, Banheiro Acessível, Quadra Esportiva, Rampas e Água Potável. Cada indicador contribui igualmente para a média final.

De onde vem os dados

Os dados de infraestrutura escolar são provenientes do Censo Escolar, realizado anualmente pelo INEP (Instituto Nacional de Estudos e Pesquisas Educacionais Anísio Teixeira). São utilizados os microdados oficiais dos anos de 2024 e 2025, com foco nos estados da região Nordeste.

Como contribuir com avaliações

Ao entrar na página de detalhes de uma escola, você pode avaliar a instituição com uma nota de 1 a 5 estrelas e deixar um comentário. Se estiver fisicamente próximo a escola (até 500m), um selo de Local Verificado será aplicado automaticamente a sua avaliação. Sua avaliação contribui para a transparência e auditoria cidadã das escolas públicas do Nordeste.
